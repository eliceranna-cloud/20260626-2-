## 1. 브랜드 철학

### 브랜드 포지셔닝
JYS 컴퍼니는 **"작지만 강한 브랜드"** 를 만드는 파트너입니다.  
화려한 외형보다 **신뢰·실용·성과**를 핵심으로 하는 브랜드 정체성을 갖습니다.

### 브랜드 키워드

```
TRUST      신뢰 — 약속을 지키고 결과로 증명하는 파트너
SIMPLE     명료 — 복잡한 마케팅을 쉽고 직관적으로 전달
GROWTH     성장 — 고객의 매출과 인지도를 함께 키우는 조력자
LOCAL      현장 — 대한민국 현장의 언어와 감각으로 소통
```

### 브랜드 퍼스낼리티

| 속성 | 설명 |
|------|------|
| **친근하지만 전문적인** | 어렵지 않게 다가오되, 실력은 믿을 수 있는 |
| **적극적이고 솔직한** | 좋은 말만 하지 않고 데이터로 현실을 말하는 |
| **꼼꼼하고 성실한** | 납기와 약속을 철저히 지키는 |
| **현실적인 낙관주의** | 어려운 상황에서도 해결책을 찾아내는 |

## 2. 로고 시스템

### 2-1. 기본 로고 구조

```
┌─────────────────────────────────┐
│                                 │
│   [J] JYS 컴퍼니                │
│       MARKETING AGENCY          │
│                                 │
└─────────────────────────────────┘

- 심볼(이니셜 J) + 한글 로고타입 + 영문 서브 태그라인 조합
- 심볼은 단독 사용 가능 (아이콘, 프로필 이미지 등)
```

### 2-2. 로고 사용 배경 규정

| 배경 | 사용 로고 | 비고 |
|------|-----------|------|
| 흰색 배경 | 기본 컬러 로고 | 최우선 사용 |
| 밝은 배경 | 기본 컬러 로고 | 명도 차이 확인 필수 |
| 어두운 배경 | 화이트 로고 | 역전 버전 사용 |
| 컬러 배경 | 화이트 또는 블랙 로고 | 가독성 우선 판단 |
| 사진 위 | 화이트 또는 반투명 배경 처리 | 단독 사용 지양 |

### 2-3. 로고 여백(클리어 스페이스)

```
최소 여백 = 로고 높이(H)의 1/2

     ←─── H/2 ───→
  ↑
 H/2
  ↓
  [     로고     ]
  ↑
 H/2
  ↓
```

> 클리어 스페이스 내에는 어떠한 텍스트·이미지·그래픽도 배치하지 않습니다.

### 2-4. 최소 사용 크기

| 매체 | 최소 크기 |
|------|-----------|
| 인쇄물 | 너비 20mm 이상 |
| 디지털 화면 | 너비 80px 이상 |
| 파비콘 / 아이콘 | 심볼 단독 사용 (32×32px~) |

### 3-3. 컬러 사용 비율

```
Primary (JYS Navy)   ████████████████░░░░   60%
Secondary (Orange)   ████░░░░░░░░░░░░░░░░   10%

```
✅ 권장 조합
  - Navy 배경 + White 텍스트

  - Orange 배경 + White 텍스트 (접근성 미달)
  - Navy + Dark Green (계열 충돌)
  - 임의 혼합 배경 그라디언트

## 4. 타이포그래피

### 4-1. 한국어 폰트

#### 메인 서체 — **Pretendard**
```
용도    : 제목, 본문, UI 전반
특징    : 가독성 높은 모던 고딕, 웹·인쇄 모두 최적
라이선스: 오픈 폰트 (무료 상업 사용 가능)
다운로드: https://github.com/orioncactus/pretendard
```

| 구분 | Weight | 크기 (디지털 기준) | 용도 |
|------|--------|-------------------|------|
| H1 대제목 | Bold (700) | 40–48px | 메인 배너, 섹션 타이틀 |
| H2 중제목 | SemiBold (600) | 28–32px | 서브 섹션, 카드 제목 |
| H3 소제목 | Medium (500) | 20–24px | 내용 구분, 리스트 제목 |
| Body 본문 | Regular (400) | 15–17px | 일반 텍스트 |
| Caption 캡션 | Light (300) | 12–13px | 주석, 출처, 날짜 |

### 4-2. 영문 폰트

#### 영문 서체 — **Inter**
```
용도    : 영문 텍스트, 숫자, 데이터 표시
특징    : Pretendard와 시각적 조화 최적
라이선스: 오픈 폰트 (무료 상업 사용 가능)
```

### 4-3. 타이포그래피 규칙

```
줄 간격 (Line Height)
  - 제목류 : 1.2 – 1.4
  - 본문류 : 1.6 – 1.8

자간 (Letter Spacing)
  - 제목    : -0.5 ~ 0 (약간 좁게)
  - 본문    : 0 ~ +0.3 (자연스럽게)

단락 간격 (Paragraph Spacing)
  - 본문 단락 : 줄 간격의 0.75배

최대 본문 너비 : 680px (가독성 한계)
```

## 5. 브랜드 보이스 & 톤

### 5-1. 브랜드 보이스 정의

JYS 컴퍼니는 **"옆집 마케팅 전문가"** 처럼 말합니다.

> 어렵지 않고, 형식적이지 않고, 그렇다고 가볍지도 않게.  
> 데이터와 경험에서 나오는 말을 쉬운 언어로 전달합니다.

### 5-2. 채널별 톤 가이드

| 채널 | 톤 | 예시 표현 |
|------|-----|----------|
| **블로그·리포트** | 전문적·정보적 | "3개월 운영 데이터 기준, 전환율이 2.4% 개선되었습니다." |
| **인스타그램** | 친근·경쾌 | "광고비 10만원으로 매출 올리는 법, 알려드릴게요 😊" |
| **카카오 상담** | 따뜻·신속 | "안녕하세요! 문의 주셔서 감사해요. 바로 확인해드릴게요." |
| **제안서·공문** | 공식·신뢰 | "귀사의 성장을 위한 최적의 마케팅 전략을 제안드립니다." |
| **광고 카피** | 간결·자극 | "매출 걱정, 이제 저희에게 맡기세요." |

### 5-3. 써야 할 언어 vs 피해야 할 언어

**✅ 권장 표현**
- "실제 데이터 기준으로 말씀드리면~"
- "어렵지 않아요, 함께 해봐요"
- "~하실 수 있도록 도와드릴게요"
- "성과로 증명하겠습니다"

**❌ 피해야 할 표현**
- 과도한 전문 용어 남발 (CPM, ROAS 등 설명 없이 단독 사용)
- "100% 보장" "무조건 성공" 같은 허위성 문구
- 고압적·지시적 어투 ("반드시~해야 합니다")
- 경쟁사 직접 비하 표현

## 6. 이미지 & 비주얼 방향성

### 6-1. 사진 방향성

**선호 이미지**
```
✅ 실제 현장감 있는 사진 (실제 가게, 사람, 제품)
✅ 밝고 자연스러운 자연광 톤
✅ 한국적 일상 분위기 (골목 상점, 카페, 시장 등)
✅ 작업하는 사람, 소통하는 장면
✅ 고해상도 (인쇄: 300dpi / 웹: 72dpi 이상)
```

**지양 이미지**
```
❌ 과도하게 보정된 스톡 사진 (외국인 모델 위주)
❌ 어둡고 무거운 톤
❌ 저화질, 픽셀 깨짐 이미지
❌ 워터마크 있는 이미지
❌ 경쟁사 로고·CI가 노출된 이미지
```

### 6-2. 그래픽 요소

| 요소 | 사용 방식 |
|------|-----------|
| **아이콘** | 라인형 아이콘 우선, 동일 스타일 일관성 유지 |
| **일러스트** | 플랫 일러스트 스타일, Navy·Orange 팔레트 적용 |
| **그래프·차트** | 심플하고 깔끔한 형태, 브랜드 컬러 사용 |
| **패턴·텍스처** | 과도한 사용 지양, 배경 보조 용도만 허용 |

### 6-3. 영상 가이드

```
영상 비율    : 유튜브 16:9 / 릴스·쇼츠 9:16 / 피드 1:1
자막 폰트    : Pretendard Bold, 흰색 + 검정 외곽선
로고 위치    : 우상단 (마진 16px 이상)
인트로/아웃트로 : 브랜드 컬러 기반, 2~3초 이내
배경음악     : 저작권 free 사용 (유튜브 오디오 보관함 권장)
```

## 7. 레이아웃 & 여백 원칙

### 7-1. 그리드 시스템

```
웹 기준
  - 최대 너비  : 1200px
  - 컬럼 수    : 12 columns
  - 거터 너비  : 24px
  - 마진       : 좌우 24px (모바일 16px)

인쇄물 기준 (A4)
  - 상하 여백  : 20mm
  - 좌우 여백  : 20mm
  - 컬럼 수    : 2~3단 (내용에 따라)
```

### 7-2. 여백 단위 (Spacing Scale)

| 레벨 | 크기 | 주요 용도 |
|------|------|-----------|
| XS | 4px | 아이콘-텍스트 간격 |
| S | 8px | 요소 내부 패딩 |
| M | 16px | 컴포넌트 간 기본 간격 |
| L | 32px | 섹션 내 그룹 간격 |
| XL | 64px | 섹션 간 간격 |
| 2XL | 120px | 페이지 최상단/최하단 |

### 7-3. 카드 컴포넌트 기준

```
배경색    : White 또는 Light Blue (#EBF2FA)
테두리    : 없음 또는 1px #E0E0E0
모서리    : 8px radius
그림자    : box-shadow 0 2px 12px rgba(0,0,0,0.08)
내부 패딩 : 24px
```

## 9. 금지 사항 (Don'ts)

### 로고 관련

```
❌ 로고 색상을 임의로 변경
❌ 로고 요소를 분리해서 사용 (심볼만, 텍스트만 = 지정 케이스 외 금지)
❌ 로고에 그림자·외곽선·그라디언트 효과 추가
❌ 로고 비율 변형 (가로·세로 비율 유지 필수)
❌ 복잡한 배경에 가독성 없는 로고 배치
❌ 최소 사용 크기 미만으로 축소 사용
```

### 컬러 관련

```
❌ 브랜드 가이드에 없는 색상을 주요 색상으로 사용
❌ 명도 대비 4.5:1 미만 조합 (웹 접근성 기준 위반)
❌ 무분별한 그라디언트 적용
❌ 3가지 이상 컬러를 동등하게 혼용
```

### 폰트 관련

```
❌ 가이드에 없는 폰트 사용
❌ 장평·자간을 과도하게 변형 (±20% 초과)
❌ 제목에 Light(300) 이하 굵기 사용
❌ 4가지 이상 폰트 혼용
```

### 전반적 금지

```
❌ 경쟁사 또는 타 브랜드 로고와 나란히 배치
❌ 브랜드 이미지를 훼손하는 불쾌한 이미지와 함께 사용
❌ 정치적·사회적으로 민감한 내용과 연관
❌ AI 생성 이미지 출처 불명확 시 외부 공개 사용
```
